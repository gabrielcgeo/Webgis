Guia de Configuração - Como Configurar Preços
Informações Básicas
Nome do Item:
  • Projeto: Cada projeto GIS criado no sistema
  • Camada: Cada camada de dados (mapas, imagens, etc.)
  • Mapa Público: Mapas compartilhados publicamente
  • Visualização: Cada 1.000 visualizações de páginas
  • Funcionário: Cada usuário registrado no sistema
  • Servidor: Cada GB de armazenamento usado por mês
Categoria:
  • Recursos: Projetos, camadas, mapas
  • Uso: Visualizações, funcionários
  • Infraestrutura: Servidor, armazenamento
Preço Base e Tiers
Preço Base:
  • Valor cobrado para quantidades pequenas
  • Serve como referência para calcular tiers
  • Exemplo: R$ 25,00 por projeto
Sistema de Tiers:
  • 6 níveis de desconto por volume
  • Quanto mais volume, menor o preço
  • Descontos configuráveis por percentual
Como Funciona o Sistema de Tiers
Tier Quantidade Desconto Preço Final Exemplo
Tier 1 51 - 200 5% R$ 23,75 100 projetos = R$ 2.375,00
Tier 2 201 - 500 10% R$ 21,50 300 projetos = R$ 6.450,00
Tier 3 501 - 1.000 15% R$ 21,25 750 projetos = R$ 15.937,50
Tier 4 1.001 - 2.000 20% R$ 20,00 1.500 projetos = R$ 30.000,00
Tier 5 2.001 - 5.000 25% R$ 18,75 3.000 projetos = R$ 56.250,00
Tier 6 5.001 - 10.000 30% R$ 17,50 7.500 projetos = R$ 131.250,00
Tier 7+ 10.001+ 35% R$ 16,25 15.000 projetos = R$ 243.750,00
Configurações Avançadas
Fator de Peso:
  • Define a importância do item no cálculo total
  • Valores maiores = maior impacto no preço final
  • Exemplo: Projetos (peso 1.0) vs Camadas (peso 0.8)
Status Ativo:
  • Habilita/desabilita o item na cobrança
  • Útil para testes ou descontos temporários
Modos de Edição
Modo Automático:
  • Preços dos tiers são calculados automaticamente
  • Baseado nos percentuais de desconto
  • Alterações nos percentuais atualizam todos os tiers
Modo Manual:
  • Permite editar preços individualmente
  • Útil para ajustes específicos ou promoções
  • Pode ser permanente ou temporário
Passo a Passo para Configurar
Passo 1: Escolher Item
  1. Defina o nome do item (ex: Projeto)
  2. Escolha a categoria apropriada
  3. Configure o preço base inicial
Passo 2: Configurar Tiers
  1. Ajuste os percentuais de desconto
  2. Configure os limites de quantidade
  3. Verifique os preços calculados
Passo 3: Testar e Ajustar
  1. Use o simulador para testar cenários
  2. Ajuste percentuais conforme necessário
  3. Verifique a competitividade dos preços
Passo 4: Salvar e Ativar
  1. Clique em Salvar para persistir
  2. Ative o status da configuração
  3. Teste no calculador principal
💡 Dica Rápida: Selecione um item no dropdown "Nome da Configuração" para ver o preço base recomendado, depois clique em "Aplicar" para usar esse preço. Ajuste os percentuais de desconto conforme sua estratégia de negócio e use o simulador para testar diferentes cenários antes de salvar!
